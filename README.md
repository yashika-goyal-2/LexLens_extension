**LexiLens** is a Chrome Extension that uses Google's **Gemini AI** to instantly analyze Terms & Conditions, Privacy Policies, and Contracts. It helps you find hidden risks, data selling clauses, and unfair terms in seconds.

## ‚ú® Features

- **Instant Analysis:** Scans the current page text automatically.
- **AI Verdict:** Gives a clear "Safe", "Caution", or "Not Recommended" score.
- **Top 5 Risks:** Highlights specific dangerous clauses.
- **Dual Language Support:** Toggle between **English** and **Hinglish** (Bole Toh style) explanations.
- **User-Friendly:** Simple, non-legal language (2-3 lines max).

## üöÄ Installation Guide

1.  **Download/Clone** this repository to your computer.
2.  Open Google Chrome and go to `chrome://extensions/`.
3.  Enable **"Developer mode"** (toggle in the top right corner).
4.  Click **"Load unpacked"**.
5.  Select the folder where you saved these files (`LexLens_extention`).
6.  LexiLens is now installed! üéâ

## üìñ How to Use

1.  Visit any website with Terms & Conditions (e.g., a signup page).
2.  Click the **LexiLens icon** in your browser toolbar.
3.  Watch the **Smart Loader** while Gemini analyzes the text.
4.  Read the **Verdict** and **Key Points**.
5.  Use the **Toggle Switch** at the top to switch language if needed.
